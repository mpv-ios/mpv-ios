## Features

- **HDR playback** - Full HDR and Dolby Vision support (Profile 8 only)
- **Hardware decoding** - Smooth playback with hardware-accelerated decoding
- **Subtitle support** - Switch between subtitle tracks on the fly
- **WiFi transfer** - Import media wirelessly from your browser
- **Picture in Picture** - Continue watching while using other apps
- **Now Playing integration** - Control playback from the Lock Screen and Control Center
